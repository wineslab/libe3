Detailed Description

JSON encoder for E3AP PDUs.

Provides JSON encoding/decoding for development and debugging.

JSON wire format uses camelCase keys (e.g. "dAppName", "ranFunctionIdentifier") and camelCase PDU type strings (e.g. "setupRequest", "indicationMessage"). PascalCase PDU types are rejected by the decoder.

Envelope format is flat: payload fields sit alongside "type", "id", and "timestamp" at root level. Messages with a "data" wrapper are rejected.
