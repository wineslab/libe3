Detailed Description

template<typename T>
class libe3::LockFreeQueue< T >

Lock-free bounded blocking queue.

Built on an MPMC lock-free ring buffer. Blocking pop variants use an adaptive spin-wait strategy:

  1. Spin with CPU pause hints (lowest latency, nanoseconds)
  2. Thread yield (cooperative, microseconds)
  3. Short sleep (50 µs) (idle wait, avoids busy-loop when quiet)
Template Parameters
TElement type. Must be default-constructible and movable.
