Detailed Description

E3AP Protocol Types - Vendor-neutral type definitions.

This header defines all E3AP protocol types and structures used throughout the libe3 library. These types are designed to be RAN-agnostic and can be used by any RAN vendor integrating with the E3AP protocol.
