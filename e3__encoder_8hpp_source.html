27/**
28 * @brief Abstract base class for E3AP encoders
29 *
30 * This class defines the interface for encoding and decoding E3AP PDUs.
31 * The design allows for multiple encoding formats (ASN.1, JSON) while
32 * maintaining the same API.
33 *
34 * Derived classes:
35 * - ASN1E3Encoder: ASN.1 PER encoding (O-RAN standard)
36 * - JsonE3Encoder: JSON encoding (for development/debugging)
37 */
