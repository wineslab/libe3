Detailed Description

Lock-free bounded queue for E3AP PDUs.

Provides the same API as the previous mutex-based implementation but uses an MPMC lock-free ring buffer internally. Blocking pop variants use an adaptive spin-wait strategy:

  1. Spin with CPU pause hints (lowest latency, nanoseconds)
  2. Thread yield (cooperative, microseconds)
  3. Short sleep (50 µs) (idle wait, avoids busy-loop when quiet)
