◆ e3_service_model_create_from_c()

e3_service_model_handle_t * e3_service_model_create_from_c ( const e3_c_service_model_desc_t *  desc)

Create a ServiceModel implementation backed by C callbacks.

The returned pointer owns a C++ ServiceModel instance. The caller retains ownership until the model is registered with an agent via e3_agent_register_sm, at which point ownership is transferred to the agent and e3_service_model_destroy will no longer free the object.

Parameters
descDescriptor containing metadata and callback function pointers
Returns
e3_service_model_handle_t* on success, NULL on allocation error
