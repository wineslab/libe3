Detailed Description

Abstract base class for E3AP transport connectors.

This class defines the transport-agnostic interface that all connectors must implement. The design mirrors the original C implementation's function pointer approach, but uses virtual methods for type safety.

Derived classes:

  • PosixE3Connector: POSIX socket-based transport (TCP, SCTP, IPC)
  • ZmqE3Connector: ZeroMQ-based transport
