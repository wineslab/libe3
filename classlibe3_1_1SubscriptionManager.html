Detailed Description

E3 Subscription Manager.

Thread-safe manager for dApp registrations and RAN function subscriptions. Handles:

  • dApp registration from E3 Setup requests
  • Subscription management from E3 Subscription requests
  • RAN function lifecycle coordination

This class is ported from the original C e3_subscription_manager with modern C++ idioms for thread safety and memory management.
