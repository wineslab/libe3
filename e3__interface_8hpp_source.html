51/**
52 * @brief E3Interface - Internal protocol coordination
53 *
54 * This class is the internal coordination layer that:
55 * - Manages E3AP lifecycle (Setup, Subscription, Control, Release)
56 * - Owns inbound/outbound processing threads
57 * - Coordinates between connector, encoder, and subscription manager
58 * - Supports simulation mode for testing without real RAN
59 *
60 * This class is NOT directly exposed to library users. Users interact
61 * with the E3Agent facade which owns an E3Interface instance.
62 */
