39/**
40 * @brief E3Interface - Internal protocol coordination
41 *
42 * This class is the internal coordination layer that:
43 * - Manages E3AP lifecycle (Setup, Subscription, Control, Release)
44 * - Owns inbound/outbound processing threads
45 * - Coordinates between connector, encoder, and subscription manager
46 * - Supports simulation mode for testing without real RAN
47 *
48 * This class is NOT directly exposed to library users. Users interact
49 * with the E3Agent facade which owns an E3Interface instance.
50 */
