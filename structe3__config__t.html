Detailed Description

Configuration for creating an E3Agent from C.

All string pointers are copied by the library; the caller may free them after the call. Use NULL for optional strings to keep library defaults.

Enum values: link_layer: 0=ZMQ, 1=POSIX. transport_layer: 0=SCTP, 1=TCP, 2=IPC. encoding: 0=ASN1, 1=JSON. Use -1 for link_layer, transport_layer, encoding, log_level to keep defaults. Use 0 for setup_port, subscriber_port, publisher_port to keep defaults.
