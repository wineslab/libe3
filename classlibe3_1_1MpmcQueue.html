Detailed Description

template<typename T>
class libe3::MpmcQueue< T >

Lock-free bounded MPMC ring buffer.

Template Parameters
TElement type. Must be default-constructible and movable.

Usage:

MpmcQueue<MyMsg> q(256);
// producer:
if (!q.try_push(std::move(msg))) { // queue full }
// consumer:
MyMsg out;
if (q.try_pop(out)) { // use out }
